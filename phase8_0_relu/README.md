## Quick Start

python src/step_8_0_relu.py

## What This Phase Teaches

- ReLU: max(0, x) — 最常用的激活函数
- Dead neurons: 负输入导致的零梯度问题
- 稀疏性: 大部分神经元输出 0

## New in Phase 8.0

- **ReLU 深入分析**: dead neuron detection
- **激活热图**: 可视化哪些神经元被激活
